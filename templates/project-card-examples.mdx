# Project Card Template Examples

Import the ProjectCard component and use it with different configurations:

```jsx
import ProjectCard from '@/components/ProjectCard'
```

## Active Project Example

```jsx
<ProjectCard
  title="HomeLink Retrofit"
  timeline="2025 Q3-Q4"
  progress={60}
  description="Retrofitting garage door integration into E83 platform. Working on CAN message decoding and module programming."
  nextMilestones="Module installation, CAN gateway configuration, testing"
  href="/projects/homelink-retrofit"
  status="active"
  color="blue"
/>
```

## Planning Phase Project Example

<ProjectCard

## Completed Project Example

```jsx
<ProjectCard
  title="CAS3 Retrofit & Comfort Access"
  description="Push-button start and keyless entry retrofit with full OEM integration."
  href="/projects/cas3-retrofit"
  status="completed"
  domain="Automotive"
  completedDate="2025 Q2"
  color="emerald"
/>
```

## Props Reference

### Required Props
- `title` (string) - Project name
- `description` (string) - Brief project description

### Optional Props
- `timeline` (string) - Project timeline (e.g., "2025 Q3-Q4")
- `progress` (number) - Completion percentage (0-100)
- `nextMilestones` (string) - Upcoming milestones or next steps
- `href` (string) - Link to detailed project page
- `status` (string) - Project status: "active", "planning", or "completed"
- `domain` (string) - Project domain: "Automotive", "Software", "Mechanical", "Electronics"
- `completedDate` (string) - Completion date for finished projects
- `color` (string) - Color scheme: "blue", "green", "purple", "emerald"

## Usage in Grid Layouts

### 2-Column Grid
```jsx
<div className="grid gap-6 md:grid-cols-2">
  <ProjectCard {...activeProject1} />
  <ProjectCard {...activeProject2} />
</div>
```

### 3-Column Grid
```jsx
<div className="grid gap-6 md:grid-cols-3">
  <ProjectCard {...completedProject1} />
  <ProjectCard {...completedProject2} />
  <ProjectCard {...completedProject3} />
</div>
```

## Color Scheme Guidelines

- **Blue**: General active projects, software-focused
- **Green**: High-progress projects, system monitoring
- **Purple**: Planning/research phase, AI/ML projects  
- **Emerald**: Completed projects (consistent theming)

## Status Badge Behavior

- **Active**: Animated pulsing dot, shows progress bar
- **Planning**: Animated pulsing dot, shows progress bar
- **Completed**: Static dot, shows domain badge instead of progress

This template provides a consistent, reusable way to display project information across the portfolio with proper theming and responsive design.
