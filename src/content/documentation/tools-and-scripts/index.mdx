
# Tools & Scripts Directory

A collection of custom scripts for project automation, validation, and workflow enhancements.

---

## [`fuckai.sh`](/documentation/tools-and-scripts/fuckai)
**Purpose:**
Lightweight emoji linting script for detecting emoji usage in project files. Primarily intended for validating AI-generated mock data and documentation.

**Features:**
* Configurable file type filtering (include or exclude modes)
* Supports ignore patterns via `.fuckai` file
* Fast, predictable output for automation and pre-commit hooks
* Reports files and lines containing emojis with totals
