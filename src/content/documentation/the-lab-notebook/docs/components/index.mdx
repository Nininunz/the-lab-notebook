# Component Documentation

This section contains documentation for all reusable components in The Lab Notebook project.

## Available Components

- [FeatureCard Component](/documentation/the-lab-notebook/docs/components/feature-card) - Interactive project showcase cards
- [LightboxImage Component](/documentation/the-lab-notebook/docs/components/lightbox-image) - Images with lightbox functionality
- [PhotoGrid Component](/documentation/the-lab-notebook/docs/components/photo-grid) - Responsive image grids
- [InfoCard Component](/documentation/the-lab-notebook/docs/components/info-card) - Informational cards
- [QuickLinkCard Component](/documentation/the-lab-notebook/docs/components/quick-link-card) - Quick access link cards
- [SectionTitle Component](/documentation/the-lab-notebook/docs/components/section-title) - Section header titles
- [ListLinks Component](/documentation/the-lab-notebook/docs/components/list-links) - Lists of links
- [LandingHeader Component](/documentation/the-lab-notebook/docs/components/landing-header) - Header for landing pages
- [ProjectCardSmall Component](/documentation/the-lab-notebook/docs/components/project-card-small) - Compact project cards
- [ProjectCardLarge Component](/documentation/the-lab-notebook/docs/components/project-card-large) - Large project cards
- [UpdateCard Component](/documentation/the-lab-notebook/docs/components/update-card) - Cards for updates and news

## Component Guidelines

Components in this project follow these principles:

- **Reusable** - Can be used across multiple pages/projects
- **Accessible** - Include proper ARIA labels and keyboard navigation
- **Responsive** - Mobile-first design with Tailwind CSS
- **Well-documented** - Clear prop descriptions and usage examples

For development guidelines, see the [Development Guide](/documentation/the-lab-notebook/docs/guides/development).
