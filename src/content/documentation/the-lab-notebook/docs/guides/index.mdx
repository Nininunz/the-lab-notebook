## Quick Start

1. **Clone and install:**
   ```bash
   git clone <repository-url>
   cd the-lab-notebook
   npm install
   ```

2. **Start development:**
   ```bash
   npm run dev
   ```

3. **Start exploring:**
   - Check out existing projects in `content/project/`
   - Read the development guide for coding standards
   - Run `npm run check` to verify your setup
