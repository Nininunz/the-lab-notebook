# Configuration

Learn how to configure your project settings and customize the behavior to fit your needs.

## Theme Configuration

The `theme.config.jsx` file contains all the theme-specific settings:

```jsx
const config = {
  logo: <span>My Documentation</span>,
  project: {
    link: 'https://github.com/yourusername/nextra'
  },
  footer: {
    text: `MIT ${new Date().getFullYear()} Â© My Documentation.`
  }
}
```

## Navigation Setup

Use `_meta.json` files to define your navigation structure:

```json
{
  "index": "Introduction",
  "getting-started": "Getting Started",
  "guides": "Guides"
}
```

## Environment Variables

Set up your environment variables for different deployment environments:

- `NEXT_PUBLIC_BASE_URL` - Your site's base URL
- `NODE_ENV` - Development or production mode

## Build Configuration

Customize your build process in `next.config.mjs`:

```javascript
export default withNextra({
  eslint: {
    ignoreDuringBuilds: false,
  },
})
```
