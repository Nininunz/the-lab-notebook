# Getting Started

Welcome to the getting started guide! This page will help you set up and begin using our documentation.

## Installation

To get started, you'll need to install the required dependencies:

```bash
npm install
# or
pnpm install
```

## Quick Start

Here's how to quickly get up and running:

1. **Clone the repository**
2. **Install dependencies** 
3. **Start the development server**

```bash
pnpm dev
```
