# Changelog

All notable changes to The Lab Notebook are documented on this page.

## Recent Updates

### 2025-01-09

#### Major Refactor: Project Structure & Component System
- **Complete site architecture overhaul** - Moved from monolithic landing page to modular component system
- **New project card system** - Created reusable `ProjectCard` and `CompletedProjectCard` components with flexible color schemes
- **Content reorganization** - Restructured navigation with dedicated pages for Projects, Docs & Guides, and Shortcuts
- **Component templates** - Added usage documentation and examples for all reusable components

#### Navigation & Content
- **Separated project portfolio** - Distinguished between active projects and completed case studies
- **Enhanced Quick Links** - Reorganized shortcuts into Development Tools, Automotive Resources, and Electronics sections
- **Documentation restructure** - Moved technical guides to `/docs-guides/` with better categorization
- **Improved visual hierarchy** - Consistent styling across project cards with hover effects and responsive design

#### Component Features
- **Flexible color schemes** - Decoupled colors from domain categories for better design control
- **Status indicators** - Added animated status badges for active/planning/completed projects
- **Progress tracking** - Visual progress bars for active projects
- **Domain categorization** - Clean domain badges without emoji clutter
- **Responsive layouts** - Grid and list layouts that adapt to screen size
