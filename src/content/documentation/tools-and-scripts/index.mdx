
# Useful Scripts Directory

A collection of custom scripts for project automation, validation, and workflow enhancements.

---

## `fuckai.sh`
**Purpose:**  
Searches project files for emoji characters, helping to enforce a “no emoji” policy or just for fun.  
**Features:**
- Scans all code and content files, excluding common directories
- Supports ignore patterns via `.fuckai` file
- Reports files and lines containing emojis, with totals
